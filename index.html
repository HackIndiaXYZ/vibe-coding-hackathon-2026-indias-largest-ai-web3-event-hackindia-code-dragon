BUILT BY

Student & Creative Developer — Lucknow, India
Built by Ojas, a developer from Lucknow, Uttar Pradesh who codes at the intersection of AI systems and cinematic UI design.

REN is an advanced interactive coding tutor and technical mentor system. It is designed to change how we learn programming: instead of just delivering massive blocks of copy-pasteable code, it segments, details, and explains every functional component of the architecture, prompting user engagement and critical thinking.

The system features persistent memory analytics, automated code segment parsing, and a high-performance web search integration — all hand-crafted in Python.
